1. Accessing the application

This section explains how the user can access the application.

  1. Umbraco Back Office

This below image get displayed once content manager authenticated with the Umbraco username and password. It contains Scottybons Project Hierarchy. The root node is “ScottyBons Base”, the child nodes “EN” and “NL” are the language nodes. The Utilities & Snippets are global items, which are used to author the content.

 

Figure 1: Umbraco Back Office

We’ll have multiple sections in Umbraco which can be shown on the left side

  1. Content: This section is used to add/ delete pages and this is where we’ll add/edit the content for the page properties.
  2. Media: This section is used to maintain the resources of the website like we can add Images, files etc.
  3. Settings: This section is for developers where we’ll have css, JavaScript files, templates and document type’s available.
  4. Developer: This section is for developers where we’ll create data types and create XSLT files.

The Footer has three columns and each column has multiple links and those properties differ.   1. FAQ Question and Answer Promos

The frequently asked questions have two properties, those are Question and Answer. The below image describes the adding the FAQs. The content manager/editor can use below properties to publish the content of FAQ Page using following steps

  1. Select "FAQ Promos"
  2. Right click the node and click "Create" option
  3. Identify the appropriate Item, select/click the "FAQ Questions"
  4. Add Heading, FAQ Question and FAQ Answer
  5. Click "Save and Publish" button

  1. Our Story Promos

Our story page contains multiple stories. The content manager/editor can use below properties to publish the multiple stories of our story page using below steps.

  1. Select "Our Story Promos"
  2. Right click the node and click "Create" option
  3. Identify the appropriate Item, select/click the "Our Story Snippet"
  4. Add page content using available properties, which are Title, Story Image and Description
  5. Click "save and publish" button to publish the content to Our Story Page.

  1. Joining Scottybons Promos (Vacancies)        

The Joining Scottybons screen list all available options in the form of question and answer format. The content manager/editor can use below properties to publish the content of Vacancies Page using following steps.

  1. Select "Joining Scottybons Promos"
  2. Right click the node and click "Create" option
  3. Identify the appropriate Item, select/click the "Joining Scottybons Questions"
  4. Add Heading, Question and Answer
  5. Click "save and publish" button to add the content in Joining Scottybons page.

  1. How to publish the changes in Umbraco Back office

  • Right Click the Utils & Snippet and click "Publish" option
  • Check Publish Utils & Snippets and all its subpages
  • Check to include unpublished child pages.
  • Click “publish" button to publish the pages.
  • Follow the same steps for “ScottyBons Base”
